# GrowSlopPatcher

This project patches any Growtopia Android apk with your desired game modification. The project basically automates what you can do in combination with apktool, ubersigner and adb on PC.

**DISCLAIMER:** This project is almost entirely gen-AI (Gemini free tier). This is not production grade and never will be, I refuse to call anything production grade unless it is made entirely by humans. I am aware how to perform what the app does with command-line tools and could orchestrate a build script for this, but Android users will expect a native app for this process and I have no real knowledge in Java or Kotlin. This app was made as a bridge for those not comfortable with command-line tools.
